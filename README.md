# 2D-GPU Graphic Accelerator for PYNQ-Z2

This project implements a 2D hardware graphics accelerator on the PYNQ-Z2 board.

## Structure
- `hdl/`: Verilog source files for the GPU components.
- `sim/`: Testbenches for functional verification.
- `pynq/`: Python drivers for interacting with the hardware.
- `ip_repo/`: Packaged IP cores for Vivado.
- `scripts/`: Automation scripts (Tcl).
- `notebooks/`: Jupyter Notebooks for demonstration.
